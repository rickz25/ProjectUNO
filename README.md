## Project Name
ProjectUNO (Desktop App)

## Overview
ProjectUNO is an internal automation system deployed across Ayala Malls tenants. It manages scheduled data synchronization between tenant systems and the central mall server via Tenant API integration.

The system is responsible for monitoring tenant connectivity (online/offline status), triggering scheduled processes, and validating CSV file uploads from individual tenants to ensure accurate and timely data transmission.
